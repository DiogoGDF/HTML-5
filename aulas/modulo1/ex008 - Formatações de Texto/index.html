Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag (não semântica).

Texto marcado

Podemos criar também um texto marcado de marca texto usando a tag MARK.

Texto deletado / texto inserido

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.O texto sublinhado é chamado de "inserido", porque é como se fosse uma lacuna e algo escrito dentro da lacuna.
